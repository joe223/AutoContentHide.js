## AutoContentHide.js


![示例](./example.png)

1. Load this `javascript` file
2. Write this in `<script></script>`

```javascript
//eg:
var AutoContentHide = new AutoContentHide({
    'LIMIT_PAR_LEN_MIN':1,
    'LIMIT_PAR_LEN_MAX':5,
    'LIMIT_PROPORTION':1.0,
    'LIMIT_PAR':10,
});

AutoContentHide.AutoHide('hide'); //element Id
AutoContentHide.AutoHide('hide2');
```

That is all
